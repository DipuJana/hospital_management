# Hospital Management System

## Overview

A Spring Boot backend application for managing hospital data such as patient records. The project follows a layered architecture to ensure clean code structure and maintainability.

## Tech Stack

* Java 17
* Spring Boot
* Spring Data JPA (Hibernate)
* H2 Database
* Maven

## Current Status

* Patient entity with Gender enum
* JPA repository setup
* H2 in-memory database configuration
* Application running successfully

## Planned Features

* CRUD APIs for Patients, Doctors, and Appointments
* Validation and global exception handling
* Pagination and sorting
* MySQL integration
* Logging
* JWT authentication and authorization

## Run the Application

### Windows
mvnw.cmd spring-boot:run

### Mac/Linux
./mvnw spring-boot:run
